# Simple-Pic 用户使用手册

## 快速开始

### 首次登录

1. 启动应用程序后，查看控制台输出的管理员密码
2. 访问登录页面：http://localhost:8080/login.html
3. 使用以下凭据登录：
   - 用户名：`admin`
   - 密码：控制台显示的随机密码

### 访问各页面

- **上传页面**: http://localhost:8080/upload.html
- **管理后台**: http://localhost:8080/admin/dashboard.html

---

## 图片上传

### 方式一：点击选择

1. 点击上传区域的任意位置
2. 在弹出的文件选择对话框中选择图片
3. 支持多选，一次可上传多张图片

### 方式二：拖拽上传

1. 将图片文件拖拽到上传区域
2. 松开鼠标即可开始上传

### 方式三：剪贴板粘贴

1. 复制图片（Ctrl+C 或右键复制）
2. 在上传页面按 Ctrl+V 粘贴
3. 图片会自动上传

### 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- SVG (.svg)

### 链接格式

上传成功后，链接会自动复制到剪贴板。可通过页面顶部切换不同格式：

- **Markdown**: `![图片名](图片URL)`
- **直链**: 图片的直接URL
- **HTML**: `<img src="图片URL" />`
- **BBCode**: `[img]图片URL[/img]`

## 管理后台

### 仪表盘

访问路径：http://localhost:8080/admin/dashboard.html

仪表盘显示：
- 存储空间统计（使用量、图片数量等）
- 活跃会话数
- 系统基本信息

### 图片管理

访问路径：http://localhost:8080/admin/images.html

**功能**：
- 按目录层级浏览图片
- 查看图片缩略图
- 删除单个图片
- 后台上传图片
- 创建/删除目录

### 存储空间管理

访问路径：http://localhost:8080/admin/storage.html

**功能**：
- 查看所有存储空间
- 创建新存储空间
- 编辑存储空间配置（路径、大小、域名）
- 删除存储空间

**存储空间配置**：
- 名称：唯一标识符
- 路径：本地文件系统路径
- 大小限制：支持 KB/MB/GB 单位
- 域名：图片URL前缀

### 用户管理

访问路径：http://localhost:8080/admin/users.html

**功能**：
- 查看所有用户
- 创建新用户
- 编辑用户信息（密码、角色、存储空间）
- 删除用户

**用户角色**：
- **USER**: 普通用户，只能上传图片
- **ADMIN**: 管理员，可访问所有管理功能

### 系统设置

访问路径：http://localhost:8080/admin/system.html

**基础信息**：
- 系统名称
- 系统描述

**水印设置**：
- 启用/禁用水印
- 水印类型（文字/图片）
- 水印内容
- 水印位置（左上/右上/左下/右下/居中）
- 水印透明度（0-1）

**安全设置**：
- 启用/禁用IP限流
- 最大请求数
- 时间窗口（秒）

**前端设置**：
- 主题（深色/浅色）
- 每页显示条数

### API 密钥管理

访问路径：http://localhost:8080/admin/apikey.html

**功能**：
- 查看所有 API 密钥
- 生成新的 API 密钥
- 删除 API 密钥

**使用 API 密钥**：
```bash
curl -X POST http://localhost:8080/api/upload \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -F "file=@image.jpg"
```

## 常见问题

### Q: 忘记管理员密码怎么办？

A: 删除 `config.yml` 文件，重启应用程序。系统会重新生成随机密码并显示在控制台。

### Q: 如何修改存储空间大小？

A: 登录管理后台，进入"存储"页面，点击对应存储空间的编辑按钮。

### Q: 上传失败提示"存储空间配额已满"怎么办？

A: 在管理后台查看存储空间使用情况，可以增加配额或清理旧图片。

### Q: 缩略图无法显示怎么办？

A: 系统会在每天凌晨3点自动生成缺失的缩略图，或者直接访问原图URL。

### Q: API 上传返回"Invalid API token"错误？

A: 检查 API 密钥是否正确，确保在管理后台生成了有效的 API 密钥。

### Q: 如何更改登录密码？

A: 登录管理后台，进入"用户"页面，编辑用户信息并设置新密码。

## 文件存储结构

图片按以下结构存储：

```
{存储空间路径}/{年份}/{月份}/{UUID}.{后缀}
```

例如：
```
./storage/default/2026/03/abc123def456.jpg
```

缩略图存储在：
```
./storage/default/.thumbnails/2026/03/abc123def456.jpg
```

## 配置文件

配置文件 `config.yml` 位于项目根目录，包含：

- 系统基础信息
- 存储空间配置
- 用户配置（密码已加密）
- API 密钥
- 水印设置
- 安全设置
- 前端配置

⚠️ **注意**：修改配置文件后需要重启应用程序才能生效（建议通过管理后台修改）。
